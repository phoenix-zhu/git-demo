#### Add a sub module
```
git submodule add REPO_URL PATH
```
config file is `.gitmodules` in repo root dir.

#### Update submodules
```
git submodule update --init --recursive
```

#### Clone a repo with submodules
```
git clone --recurse-submodules REPO_URL
```
